# SISTEM MANAJEMEN BUKU

Sistem Manajemen Buku (BookManager) adalah web responsif yang dibuat untuk mengelola koleksi buku secara digital. Web ini terdiri dari 3 halaman utama yang saling terhubung melalui navigasi, menggunakan HTML5, CSS, Bootstrap, dan JavaScript vanilla. Seluruh tampilan dibuat agar pengguna nyaman melihat di berbagai ukuran layar, mulai dari desktop sampai smartphone.

### 1. index.html — Halaman Beranda
Halaman ini merupakan halaman utama web. Ketika pengguna pertama kali membuka web, mereka akan disambut oleh tampilan hero section yang menampilkan judul besar dan ajakan untuk mulai menambah buku.

### 2. books.html — Halaman Koleksi Buku
Halaman ini menampilkan daftar buku yang tersedia dalam bentuk (card) yang tersusun dalam grid responsif. Setiap kartu memuat informasi tentang satu buku.

### 3. add-book.html — Halaman Tambah Buku
Halaman ini berisi formulir untuk menambahkan data buku baru ke dalam sistem. Formulir dilengkapi dengan validasi input secara real-time maupun saat dikirim.
